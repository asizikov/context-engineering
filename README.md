# Context Engineering

An interactive tutorial on context engineering for AI coding agents — covering foundational principles and practical techniques for designing, structuring, and delivering the right context.

**[Open the tutorial →](https://blog.cloud-eng.nl/context-engineering/)**

## What you'll learn

- What context is and why it matters for AI agents
- How the agent loop works and how context grows
- Why context degrades over time (context rot)
- Techniques for keeping context focused and effective
- How to use memory, compaction, AGENTS.md, and observability
